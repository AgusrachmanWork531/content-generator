# Workflow: Test And Validate

Gunakan workflow ini setelah perubahan kode.

## Steps

1. Cek package manager.
2. Cek command test/lint/build.
3. Jalankan validasi yang aman.
4. Catat hasil.
5. Jika gagal, analisis penyebab.
