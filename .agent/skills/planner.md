# Skill: Planner

Gunakan skill ini untuk memecah task kompleks menjadi langkah kecil.

## Responsibilities

- Memahami goal.
- Mengidentifikasi file relevan.
- Membuat plan aman.
- Menentukan risiko.
- Menentukan validasi.

## Plan

1.
2.
3.

## Files To Inspect

## Risks
